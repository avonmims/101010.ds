Projects

  • Telecommunications Statistical Data Analysis: Surf vs Ultimate An in depth analysis on two cell phone plans, Surf and Ultimate, with the goal of observing revenue and possible behavioral patterns in customers. A Jupyter file converted to HTML for display purposes. This was one of my first projects in the data science program, and the very first project I would perform hypothesis testing.
  • Video Game Market Analysis (1980-2016) A financial analysis on videogame titles and their sales performance. Looking through multiple lenses to see the performance by genre, rating, console, and country.
  • Machine Learning for Personalized Cell Plan Recommendations Utilizing two cell phone plans to train a binary classification model. This model suggests the optimal plan based on user behavior, aiming for an accuracy exceeding 75% while outperforming baseline naive predictions.
  • Bank Customer Churn Prediction Using Machine Learning This project is a binary classification machine learning task designed to predict whether bank customers will stay or leave (churn) based on their credit score, geography, age, tenure, balance, and product usage. Model performance was measured using the F1 score, with a target threshold of 0.59 on the test set.
  • Machine Learning for Oil Well Profitability & Risk Assessment Developed a predictive analytics solution to optimize oil well selection across multiple regions using linear regression. I performed thorough EDA and feature engineering on real-world geospatial datasets, built and validated regression models to forecast oil reserves, calculated break-even yields, and implemented profit estimation logic. Bootstrapping was applied to quantify profit distribution, risk of losses, and confidence intervals. The project delivered actionable recommendations based on statistical and business criteria (risk < 2.5%), automating region selection for safe, profitable investment. Tools used include Python, Pandas, scikit-learn, NumPy, and Matplotlib.
  • Gold Purification Yield Prediction & Process Optimization Led a data-driven optimization project for gold purification. Leveraged advanced data preprocessing, feature engineering, and machine learning (Random Forest, Linear Regression) to forecast output yields and validate process efficiency. Conducted rigorous data validation, handled missing values, and implemented custom sMAPE metrics for model evaluation. Delivered actionable insights for maximizing gold yield and minimizing operational risk, outperforming baseline models with (< 3.2%) error on unseen data. Tools used include Python, Pandas, scikit-learn, NumPy, and Matplotlib.
  • Used Car Price Prediction with Machine Learning For Rusty Bargain, I developed a machine learning solution to estimate used car market values using historical data (technical specs, trim, price). The project involved thorough data cleaning, feature engineering, and handling missing values. I trained and compared several models—Linear Regression, Random Forest, LightGBM, and CatBoost—evaluating them on prediction quality (RMSE), speed, and training time. LightGBM delivered the best results in both accuracy and efficiency, making it the top recommendation for real-time price predictions in the app.
  • Time Series Forecasting for Taxi Demand In this project, I built a predictive model for Sweet Lift Taxi to forecast hourly taxi orders at airports. Using historical data, I performed data resampling, exploratory analysis, and trained multiple models—including Linear Regression, AutoRegression, ARIMA, and SARIMA. After tuning hyperparameters, the SARIMA model achieved an RMSE below the target threshold of 48, enabling more accurate demand predictions during peak hours. This solution helps optimize driver allocation and improve service efficiency.
  • Customer Churn Prediction & Retention Strategy Led a data-driven project to predict customer churn for a telecom provider. Utilized advanced data preprocessing, feature engineering, and machine learning (XGBoost, CatBoost, Ensemble Voting) to identify at-risk customers and optimize retention strategies. Tackled class imbalance with custom weighting and compared multiple balancing techniques. Achieved robust model performance with an F1 Score of 0.64 and ROC-AUC of 0.85 on unseen data.
